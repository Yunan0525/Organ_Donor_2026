# CORRELATION

This folder contains microbial correlation and community structure analyses performed on intestinal microbiome datasets from human organ donors.

## Contents

### Correlation analyses

Contains scripts and outputs generated using:
- CCREPE (Compositionality Corrected by REnormalization and PErmutation)

CCREPE was used to estimate statistically significant microbial correlations while accounting for compositional effects in microbiome abundance data.

Main analyses include:
- species-species correlations
- community interaction networks
- intestinal segment-specific associations
- lumen versus mucosa comparisons

## heatmap/

Contains:
- correlation heatmaps
- processed correlation matrices
- clustering visualizations
- abundance-scaled association plots

## biplots/

Contains:
- ordination biplots
- principal coordinate analysis (PCoA) visualizations
- ecological clustering analyses
- metadata overlay visualizations

## Scripts

Contains scripts and helper files used for:
- abundance preprocessing
- compositional filtering
- CCREPE correlation analyses
- clustering analyses
- heatmap generation
- ordination and biplot visualization
