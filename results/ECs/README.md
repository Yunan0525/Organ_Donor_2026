# ECS

This folder contains enzyme functional profiling analyses derived from shotgun metagenomic sequencing data from human organ donor intestinal samples.

Functional profiling focused on Enzyme Commission (EC) annotations to investigate metabolic and functional variation across intestinal regions and ecological niches.

## Contents

### EC abundance tables

Processed EC abundance profiles generated from HUMAnN functional profiling workflows.

## Beta diversity analyses

Contains functional beta diversity analyses based on EC abundance profiles
