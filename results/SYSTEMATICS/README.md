# SYSTEMATICS

This folder contains phylogenetic and taxonomic visualization resources generated for the organ donor gut microbiome study.

The analyses in this folder were primarily used for:
- phylogenetic tree visualization
- taxonomic relationship exploration
- iTOL annotation and visualization
- comparative microbiome systematics analyses

## Contents

### iTOL annotation files

Files used for visualization in the Interactive Tree of Life (iTOL) platform

These files were generated for visualization of microbial phylogenetic relationships across intestinal regions and ecological niches.

### Tree files

Phylogenetic tree files used for iTOL visualization.

### Scripts

R scripts and helper scripts used to:
- generate iTOL annotation files
- process phylogenetic abundance matrices
- prepare metadata tracks
- generate color mappings and labels

## Visualization workflow

1. Generate abundance or taxonomy tables.
2. Create phylogenetic trees or import reference trees.
3. Generate iTOL annotation datasets using custom R scripts.
4. Upload tree and annotation files to iTOL for visualization.

## Notes

- iTOL visualizations were used for exploratory analyses and manuscript figure generation.
- Some annotation files may require manual upload to the iTOL web interface.
- Tree visualization styling was finalized interactively within iTOL.
