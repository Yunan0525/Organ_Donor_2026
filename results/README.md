# CARD Analysis

This folder contains antimicrobial resistance (AMR) analyses performed using the CARD database.

## Contents

### raw_outputs/
Raw CARD annotation outputs for all organ donor metagenomes.

### q2/
QIIME2 artifacts and visualizations:
- alpha diversity
- beta diversity
- ordination analyses

Includes:
- `.qza`
- `.qzv`

### visualization/
R scripts used for:
- AMR diversity analysis
- visualization
- statistical testing
GraphPad Prism files used for final figure generation.
